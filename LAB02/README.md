# 9. Ответы на контрольные вопросы

### 1. Что делает тема в WordPress, а что — плагин?

Тема в WordPress отвечает за внешний вид сайта. Она определяет дизайн страниц, расположение элементов, стили текста, цвета, шрифты и структуру отображения контента.

Плагин расширяет функциональность сайта. С его помощью можно добавлять новые возможности, например формы обратной связи, галереи, SEO-инструменты, системы безопасности и другие функции.

Таким образом, тема отвечает за оформление сайта, а плагины — за его дополнительную функциональность.

---

### 2. Почему при смене темы контент сайта не теряется?

Контент сайта хранится в базе данных WordPress, а не внутри темы.

Тема отвечает только за отображение данных, а сами записи, страницы, комментарии и медиафайлы сохраняются в базе данных.

Поэтому при смене темы дизайн сайта изменяется, но сам контент остаётся без изменений.

---

### 3. Как можно изменить внешний вид сайта без редактирования кода?

Изменить внешний вид сайта можно несколькими способами:

* через настройки темы в разделе «Внешний вид → Настроить»
* установив другую тему оформления
* используя визуальные конструкторы страниц
* изменяя параметры цвета, шрифтов и структуры сайта в настройках темы

Эти инструменты позволяют изменять дизайн сайта без редактирования программного кода.
